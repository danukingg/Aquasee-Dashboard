# Aquasee Dashboard (Frontend Prototype)

Proyek ini adalah prototipe dashboard monitoring kolam yang dikembangkan sebagai proyek latihan selama masa perkuliahan.

## Tech Stack
    Framework: React 19
    Build Tool: Vite
    Styling: Tailwind CSS (Custom Theme & Dark Mode)

## Catatan
Proyek ini difokuskan pada implementasi UI/UX yang presisi dan responsif. Saat ini status proyek masih dalam tahap Frontend (Client-side) tanpa keterhubungan ke Backend.

## Design Specifications:
    Typography
        Primary Font: Inter (via Google Fonts)
        Weights: Light (300), Regular (400), Bold (700)
        
    Color Palette
        Primary: #0d9be7
        Surface Dark: #1a262d
        Text Primary: #ffffff
        
    Assets
        Icons: Material Symbols / Lucide React
        Logo: By Nano Banana
